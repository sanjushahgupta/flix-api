flixApi

With this application, users can access information about movies and can save their favorites movies with their credentials.

Business Logic URL HTTP Method Path Parameters Request body data format Response body data format
Get a data about single movie by title /movies/:title GET title:String None Json object holding data about movie. {Title:String, Description: String, Genre: { Name: String, Description: String }, Director: { Name: String, Bio: String, Birth: Date, }, Actor: String, Image: String, Featured: Boolean}
Eg: {Title:"Titanic", Description:"It is a 1997 American epic romance."", Genre: { Name: "Romance", Description:"It is based on accounts of the sinking of RMS Titanic." }, Director: { Name: "James Cameron", Bio:"He is also film director and screenwriter." , Birth: "1954", }, Actor: "Leonardo DiCaprio", Image: "titanic.png", Featured: false}

Register new user /register POST None Json object holding data about user({userName:String, Password: String, Email: String, Birth: String}eg: { "userName": "Avi", "Password": "127%hj34@6", "Email":"abhi@gmail.com", "Birth": "1990-02-13"}) Json object holding data about added use, including ID and favourites movies array (id,userName,Email,Birth,favoritesMovies) eg:{ "userName": "Avi", "Email": "abhi@gmail.com", "Birth": "1990-02-13", "favoriteMovies": [], "_id": "649b15eb9c3c29ba3c3c257a", "__v": 0 }
Get genre's description about a genre by name /movies/genre/:name GET name:String None Description of specific genre. "It is a biographical drama based on the life of South Indian actress Savitri."
Get data about a director by name /movies/directors/:name GET name:String None Json object holding data about specific director(director's name,bio,birthdate) eg:{ "Name": "Sukumar", "Bio": "Sukumar is a talented South Indian filmmaker known for his unique storytelling and innovative approach to filmmaking.", "Birth": "1970-01-10T23:00:00.000Z" }
Update their user info /user/:userName PUT userName:String {userName:String, Password: String, Email: String, Birth: String} eg: { "userName": "Abhir", "Password": "127%34@6", "Birth": "1990-02-13"} Json object holding updated user's information(id,userName,Email,Birth,favoritesMovies[]) eg:{ "_id": "649b15da9c3c29ba3c3c2577", "userName": "Abhir", "Email": "abhi@gmail.com", "Birth": "1990-02-13", "favoriteMovies": [], "__v": 0 }.
Add movie to favourite list /addfab/:userName/:movieTitle POST userName:String, movieTitle:String None User's json object with new movie added(id,userName,favoritesMovies[ref:newMovie]) eg: { "_id": "649990b8cfb9285bf3b0840a", "userName": "Abhir", "favoriteMovies": [ "64996bafcfb9285bf3b083fb", ] }
Remove movie from favourite list /deletefab/:userName/:movieTitle DELETE userName:String,movieTitle:String None User's json object without requested movie i.e(userName,favoritesMovies[ref]) { "userName": "Abhir", "favoriteMovies": [ "64996bafcfb9285bf3b083fb", "64996967cfb9285bf3b083f8" ] }
Deregister existing user /deleteUser/:userName DELETE userName:String None Text message indicating a requested account has been removed. eg: Account deleted.
